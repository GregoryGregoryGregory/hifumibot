An enjoyable and helpful Discord bot

Hifumi is a Discord bot that can do things from simple moderation to music and fun commands, but that's only some features. The main focus of the bot is providing fun and useful tools for your server. Hifumi is in constant development and having the best uptime as possible.

Invite Hifumi to your server now!

Features

Search around the web from your chat.

Get quick results with searching Google, Twitch channels, Urban Dictionary, etc. and also you can search for facts about cats or everything.

Have fun commands included, all-in-one category.

Hifumi haves a lot of miscellaneous commands like 8 ball, rolling a dice, coinflip, GIF searching, chocolate giving, etc. and why not? Slap someone just in case. ¯\_(ツ)_/¯

Talk with Hifumi (cleverbot)

By using the talk command, you can have a conversation with Hifumi and talk about random things with her.

Simple moderation, faster than another!

It will only take seconds to set up welcome messages, custom permission replies and of course, apply the ban hammer to those people that became annoying for your server.

Play music!

Yeah, Hifumi can play songs too and also search on YouTube for new trending songs.

Set levels for users!

Choose the server people you want to set levels. Set levels to set up basic permissions for the bot, set level 0 for every normal user, level 1-2 for music administration and level 3 for administrators/moderators. You can also set up negative levels to tell Hifumi what people you don't want to see interacting with the bot.
